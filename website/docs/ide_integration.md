# IDE integration

You can now use `BugStalker` with your IDE thanks to DAP (Debug Adapter Protocol) support.

## Supported IDEs

- **VSCode** - via [extension](https://marketplace.visualstudio.com/items?itemName=BugStalker.bugstalker)

![demo](/gif/vs_ext_demo.gif)

- **Zed** - via [extension](https://zed.dev/extensions/bugstalker-dap)

![demo](/gif/zed_ext_demo.gif)
